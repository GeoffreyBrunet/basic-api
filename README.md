# basic-api
Basic REST API in rust, using a minimalist web framework, [Rocket.rs](https://rocket.rs/).


## Install and run project

### Install rust:
```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Build and run API
```shell
cargo run
```
